## Yahoo App vs 網頁版 vs FanGraphs 三方排名校正

### 核心發現：三套排名系統差異巨大

| 排名系統 | 來源 | 特徵 | 你的對手用哪個 |
|---------|------|------|--------------|
| FanGraphs ADP | 跨平台共識 | 最均衡 | 少數進階玩家 |
| Yahoo 網頁版 Pre-Season | 編輯手動排列 | 偏傳統評價 | 桌機選秀者 |
| **Yahoo App Projected Rank** | 預測系統自動生成 | **SP 極端偏高** | **大部分 manager** ✅ |

### App 排名的 SP 偏重現象

App 前 20 名中有 **8 個 SP**（40%）：Skubal(2)、Skenes(4)、Crochet(7)、Woo(10)、Sánchez(11)、deGrom(16)、Sale(17)、Gilbert(20)

- ✅ **好處：** SP 被搶走 → 打者掉到更低順位 → 我們的打者套利目標更容易拿到
- ❌ **壞處：** 我們的 SP 套利目標（Webb #25、Wheeler #26、Kirby #38）被大幅推高，不再是「偷到」

4. **R4 成為新的黃金地帶** → Freeman(41) + Cade Smith(43) 都在 R4，VOR 分別是 +6 和 +9

## Step 15：常見選秀分叉決策樹

### 分叉 1：「前 3 輪要不要拿 SP？」

```
第 1 輪有 Skubal 嗎？
├── 有 → 拿 Skubal（VOR +12 遙遙領先）
│        → 第 2 輪拿 Vlad/Chisholm（打者+守位）
│        → 第 3 輪補打者或再一支 SP
└── 沒有 → 先拿打者（Soto/Judge/Ramírez）
          → 第 2-3 輪看 Skenes/Crochet 有沒有滑落
          → 若 SP 都被搶光，第 3 輪拿 Gilbert/Sánchez/Woo
```

**核心邏輯：** Skubal VOR +12 是唯一值得在任何球員之前選的 SP。其他 SP（Skenes +8、Crochet +8）可以等到第 2-3 輪。

### 分叉 2：「2B 什麼時候拿？」

```
第 2 輪有 Chisholm (ADP 20) 嗎？
├── 有 → 拿（2B 選擇極少，他有 2B/3B 雙守位）
└── 沒有 → 第 3 輪 Marte (ADP 33) 還在嗎？
          ├── 在 → 拿 Marte（2B 最佳格式選項）
          └── 不在 → 不恐慌，等 Altuve (ADP 116) 或 Albies (ADP 158)
                    → 這兩人 VOR +5，和 ADP 39 的 Machado 等值
                    → 省下的 3-12 輪專選其他稀缺位+SP
```

**核心邏輯：** 2B 的套利空間巨大——Altuve 和 Albies 在 ADP 100-160 提供和前 40 ADP 球員相當的 VOR。早搶或晚拿都行，但不能完全忽略。

### 分叉 3：「CL 要搶還是用 SU 替代？」

```
你的 RP 策略偏好？
├── 穩健路線：第 3 輪拿 Miller (VOR +7) + 中段拿 Helsley/Williams
│             → 搭配 1-2 個免費 SU 覆蓋 HLD
│             → SV + HLD 兩欄都穩
└── 激進路線：跳過所有 CL → 全押 SU
              → Cade Smith (VOR +9) + Whitlock (+7) + Gaddis (+6) + Jax (+4)
              → HLD 欄碾壓，SV 欄放棄（每週 0-8）
              → 風險：SV 欄幾乎每週輸
              → 但省下的 ADP 資源可補強其他 4 個投手欄

推薦：混合路線
→ 第 3-4 輪拿 Miller 或 Cade Smith（CL 或頂級 SU 各一）
→ 第 7-9 輪拿 Helsley/Iglesias/Palencia（超值 CL 窗口）
→ 最後幾輪免費拿 Whitlock/Gaddis/Garcia（SU 免費午餐）
→ 結果：SV + HLD 都有競爭力
```

### 分叉 4：「捕手什麼時候拿？」

```
第 2 輪尾段：Cal Raleigh (ADP 18) 還在嗎？
├── 在 → ⛔ 不拿（格式陷阱，188K+.230 AVG）
└── 不論 → 三個窗口策略：

窗口一（ADP 53-55）：Ben Rice / Contreras
├── 優先 Contreras（格式最不傷的 C，+1 淨分）
└── Rice 有 C/1B 雙守位彈性

窗口二（ADP 150-155）：Rutschman
└── 格式甜蜜點（低 K、好 BB、好 AVG，不失血）

窗口三（ADP 175+）：Moreno / Kirk / Alvarez
└── 最低成本，接受 C 位產出低於平均

⚠️ 避開：Raleigh (ADP 18)、Goodman (ADP 64, K 風險)
```

### 分叉 5：「第 5-6 輪要繼續拿打者還是開始拿 SP？」

```
已拿打者數量？
├── ≤ 2 名打者 → 優先補打者（Freeman、Harper、Greene）
│                → SP 可以等到 Webb (ADP 61) / Kirby (ADP 67)
├── 3 名打者 → 這是 SP 的甜蜜點
│              → Webb (VOR +6) 和 Kirby (VOR +5) 在這裡，格式比 ADP 更高的 SP 更好
│              → 拿 1 SP + 1 打者
└── 4+ 名打者 → 必須開始拿 SP
               → Webb、Kirby、deGrom 任選
```

## 附錄 A：45 秒速查決策規則

每輪 45 秒，沒時間查表時用這套速篩：

### 打者：三步速篩

```
① K% < 20%？  → 通過（超過 20% 不選）
② BB% > 10%？ → 加分
③ SLG > .450？ → 確認有長打
→ 三項中至少兩項通過 → 選
```

- 第一刀（K%）砍掉所有格式陷阱：Schwarber (27%)、Raleigh (28%)、De La Cruz (25%)、Neto (24%)
- 第二刀（BB%）鎖定格式贏家：Soto (17%)、Vlad (13.7%)、Freeman (12%)、Betts (14%)
- 第三刀（SLG）防止只有接觸沒有長打：排除 Perdomo (.340 SLG)、Yandy Diaz (.400 SLG) 等空有低 K 但缺火力的類型

### 投手：一個數字

```
BB% < 7% → 選
BB% 7-9% → 看其他項再決定
BB% > 9% → 不選
```

投手 BB 在格式下雙重受罰（BB 欄 + WHIP 欄），控球是唯一最重要指標。

### 不確定時的保底原則

- 打者：**選低 K 的那個**（K% 每高一級，淨分平均掉 1.2）
- 投手：**選低 BB 的那個**（BB% 每高 1%，格式兩欄同時失血）
- 兩人差不多：**選 2B/3B 資格的那個**（守位稀缺加成）
