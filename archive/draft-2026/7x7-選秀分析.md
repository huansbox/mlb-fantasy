# 7×7 勝負制選秀分析（確認版）

> **格式確認（2026-03-13）**
>
> **賽制**：H2H One Win（每週贏 8/14 類別 = 1 勝）
>
> **打者 7 項**：R, HR, RBI, SB, BB, AVG, OPS
> **投手 7 項**：IP, W, K, ERA, WHIP, QS, SV+H
>
> **vs 8×8 的實際改動只有兩個：**
> 1. **移除打者 K（負向）** → 高三振球員完全解鎖
> 2. **SLG → OPS** → 加入 OBP 維度，BB 雙重計算（BB 欄 + OPS 的 OBP）

### 1.4 BB 的雙重價值

**BB 是 7×7 最有價值的單一指標**：
1. BB 直接貢獻 **BB 類別**（與 8×8 相同）
2. BB 提升 OBP → 提升 **OPS 類別**
3. BB 不算 AB → **不傷 AVG**

所以高 BB 球員在 BB 欄和 OPS 欄同時受益，且不影響 AVG。這使得 Soto（BB+2）、Judge（BB+2）、Schwarber（BB+2）在 7×7 中擁有結構性優勢。

## 四、勝負制 Punt 路線

### 4.1 14 類別勝負制

全部 14 類別（7 打 + 7 投）合計。**贏 8+ 類別 = 該週 1 勝。**

Punt 2 類別 = 給掉 2/14 → 需贏 8/12 剩餘 = 67% → 完全可行

### 4.2 推薦：Punt SB + Punt SV+H

| 類別 | 目標 | 說明 |
|------|------|------|
| R | **贏** | 高 PA 力量型自然有 R |
| HR | **贏** | 核心強項 |
| RBI | **贏** | 力量型自然有 RBI |
| SB | **放棄** | 不選速度型 |
| BB | **贏** | 高 BB 打者雙重計算 |
| AVG | **隨緣** | .260-.280 級別足夠 |
| OPS | **贏** | 力量+BB 組合 |
| IP | **贏** | 堆 SP 工作馬 |
| W | **贏** | SP 自然有 |
| K | **贏** | SP 累積 |
| ERA | **贏** | 好 SP |
| WHIP | **贏** | 好 SP |
| QS | **贏** | SP 工作馬 |
| SV+H | **放棄** | 不選 RP |

**預期每週：10-12 勝 vs 2-4 負 = 穩定拿下週勝**

### 4.4 不 punt 也可以

7×7 勝負制下，punt 有優勢但**不是必須**。如果前幾輪拿到 Witt(SB+2) 或 Acuña(SB+1)，可以不 punt SB。關鍵是：

- **不要為了 SB 犧牲 HR/RBI/OPS**
- 如果順位前段拿到兼具速度+力量的球員（Witt、Tatis、Acuña），就不用 punt
- 但如果順位中後段，power-first 是更穩的路線

## 五、8×8 → 7×7 關鍵策略調整

| # | 8×8 策略 | 7×7 確認版調整 | 原因 |
|---|---------|---------------|------|
| 1 | 低 K% 球員溢價 | **K 優勢完全消失** | K 欄不存在了 |
| 2 | Schwarber/Raleigh/Cruz 是格式陷阱 | **部分翻身**（Schwarber +3→+4） | K 懲罰消失 |
| 3 | BB 重要 | **BB 更重要**（BB + OPS 雙重計算） | OPS 的 OBP 成分讓 BB 加倍 |
| 4 | SLG 獨立計分 | **OPS 取代 SLG** | 高 OBP+高 SLG 最佳 |
| 5 | AVG 獨立計分 | **AVG 保留** | 高 AVG 仍有價值 |
| 6 | SV/HLD 各佔一欄 | **合併 SV+H** | RP 價值腰斬 |
| 7 | 投手 BB 扣分 | **BB→IP** | 工作馬 SP 暴漲，控球溢價消失 |
| 8 | 類別制（每項都重要） | **勝負制（贏 8/14 就好）** | 可 punt 2-3 項 |

## 六、結論

### vs 8×8 的真正影響

確認版 7×7 比草案溫和得多。**核心改變只有「拿掉 K」和「SLG→OPS」**，BB 和 AVG 都保留。

| 面向 | 影響 |
|------|------|
| **Judge** | ↑↑ 成為打者 VOR #1（K-1 消失） |
| **Vlad/Ramírez** | ↓ 失去 K+2 菁英優勢（仍是頂級但不再獨佔） |
| **Betts/Hoerner/Kwan** | ↓↓ K+2 消失，OPS 不突出 |
| **BB 重要性** | ↑ BB 雙重計算（BB 欄 + OPS） |
| **SP 工作馬** | ↑↑ IP 類別新增 |
| **RP** | ↓↓ IP-2 + SV+H 合併 |
| **勝負制** | 可 punt SB + SV+H |
