## 任務 B — FA 比較與升級評估

### B.1 基本勝出門檻（必要條件）

對每位 FA 候選 + 觀察中打者計算 2026 Sum（同 Pass 1 規則），vs 最弱 4 人中對應位置可替換者：

- Sum 差 ≥ 3 分（整體明顯勝）**且** 3 核心指標至少 2 項正向（每項 ≥ 0）
- `+5 +1 -3 = +3` **不算勝出**（一項大輸）
- `+1 +1 +1 = +3` 算勝出（三項都贏）
- `+2 +2 -1 = +3` 算勝出（兩項明顯勝）

不滿足 → 進入 PASS 區塊（不再評估 add 標籤）。

### B.2 ✅ Add 加分標籤（升級信心）

| 標籤 | 條件 | 意義 |
|------|------|------|
| **✅ 雙年菁英** | 2025 Sum ≥ 24（從 prior_stats 算同 Pass 1 規則） | 不是單年噪音，雙年確認 |
| **✅ 球隊主力** | 2026 PA/Team_G ≥ 3.5（若資料缺，用 prior_stats.pa_per_team_g 近似） | 球隊重視，能累積數據 |
| **✅ 近況確認** | 14d xwOBA Δ ≥ +0.035 vs season（若 14d 資料有；否則跳過此標籤） | 升溫中，買點 |

### B.3 ⚠️ Add 警示（謹慎）

| 警示 | 條件 | 意義 |
|------|------|------|
| **⚠️ 上場有限**（強警示） | 2026 PA/Team_G < 2.5（或 prior 近似） | 球隊不愛 → 數據再好也轉不成累積 |
| **⚠️ 樣本小** | 2026 BBE < 30 | 信心低，可能噪音 |
| **⚠️ Breakout 待驗** | 2025 Sum < 18 或無 prior 數據 | 單年表現，未經雙年確認 |
| **⚠️ 近況下滑** | 14d xwOBA Δ ≤ -0.035 vs season（若有資料） | 別追下滑中的 |

### B.4 升級判斷

| 條件 | 判斷 |
|------|------|
| ≥2 個 ✅ + 無 ⚠️ | **立即取代** |
| 1 個 ✅ + 無 ⚠️ 上場有限 | **取代** |
| 1 個 ✅ + ⚠️ 上場有限 | **觀察**（強警示否決升級） |
| 多個 ⚠️ | **觀察** |
| 無 ✅ | **觀察** |

優先動 urgency P1；P2-P4 需更多 ✅ 標籤才考慮。

### B.5 PA/Team_G 方向說明

- **Drop 觀點**（任務 A）：PA/TG 越高 → urgency 越高（拖累放大）
- **Add 觀點**（任務 B）：PA/TG 越高 → 信心越高（球隊重視 → 數據能轉累積）
- 兩個方向相反，因為意義不同：drop 看「每天傷害」，add 看「能否輸出」

**fa_scan 不判斷**：守位 / 角色脈絡 / 單點故障 / 邊際遞減 → 特殊情況由用戶手動處理。
