## 分歧的物理解釋

三視角看**不同時間尺度**：

| 時間尺度 | 判斷 | 誰選 |
|---------|------|------|
| **本週戰場** | López 會炸拖比率 | Agent 2 + v3 |
| **中期 2-4 週** | López 底層品質會回歸（xERA 3.74）；Nola 真實水準就 4.70 不會變好 | Agent 1 |
| **整季軌跡** | Nola 速度永久退化 → 持續惡化；López 若 Strider 歸隊可能被擠出輪值 | Agent 3 |

**沒有絕對對錯** — 看你要 H2H 週勝率還是整季穩定性。

## v3 框架的盲點總結

v3 過度集中在「當下產量 snapshot」：
- K/9 + IP/GS + BB/9 都是當下累積
