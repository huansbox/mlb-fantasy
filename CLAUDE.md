# MLB Fantasy Baseball 選秀分析專案

## 專案概述

針對 2026 Yahoo Fantasy Baseball 聯賽，進行格式專屬的選秀分析與策略制定。

## 聯賽設定

- **平台**：Yahoo Fantasy Baseball
- **賽制**：H2H **勝負制**（每週對戰，贏 4/7 類別 = 1 勝，不論贏幾項）
- **隊伍數**：12 隊，分兩個聯盟
- **選秀**：蛇形選秀（Snake Draft），22 輪，順位未定
- **名單配置**：
  - 打者：C / 1B / 2B / 3B / SS / OF×3 / UTIL×2（共 10 人）
  - 投手：SP×4 / RP×2 / P×3（共 9 人）
  - 板凳 / IL / NA
- **計分類別（7×7）**：（取代原 8×8 方案）
  - 打者：R, HR, RBI, SB, **H+BB**(累積), **K/AB**(負向比例), **OPS**(正向比例)
  - 投手：W, **SV+HLD**(合併累積), K, QS, ERA, WHIP, **IP**(累積)
- **限制**：
  - 每週最低 65 投球局數（Min IP = 65，未達 → ERA + WHIP 判負）
  - 每週最多 6 次異動（Max Acquisition = 6）

## 7×7 格式關鍵特性

### vs 原 8×8 的結構性改變

1. **H+BB 取代 BB** → 高打席+高 OBP 球員雙重受益（純 BB 型不夠，要 H 也多）
2. **K/AB 比例取代 K 計數** → 低 K% 仍有優勢但高 PA 球星不再被額外懲罰
3. **OPS 取代 AVG+SLG（2→1）** → 力量+耐心組合稱王，純接觸型（Kwan、Perdomo）崩盤
4. **IP 取代投手 BB** → 工作馬 SP 暴漲（Webb +7、Valdez +5），控球型溢價消失
5. **SV+HLD 合併** → RP 價值腰斬（覆蓋 1 類別而非 2）+ IP -2 再重創
6. **勝負制** → 贏 4/7 就夠 → 可策略性放棄（punt）1-2 個弱項

### 勝負制策略含義

- **均衡型不再是唯一解**：可選擇「力量路線」或「速度路線」，故意放棄 1-2 項
- **力量 punt 路線**：專攻 HR, RBI, OPS, H+BB，放棄 SB, K/AB（4:3 穩贏）
- **極端型球員翻身**：Alvarez（零 SB）、Schwarber（高 K）在 punt 路線下不再是負擔

## 核心分析框架

三層遞進分析：

```
Layer 1：格式類別貢獻評分（每位球員在 7 項中的 +2 到 -2 評分）
Layer 2：跨守位替代價值 VOR（相對於守位替代水平的超額貢獻）
Layer 3：ADP 套利標記（格式排名 vs 市場 ADP 的落差）
+ Layer 4：勝負制 punt 路線適配性
```

## 7×7 已確立的分析結論

### SP 價值排序（IP 類別加持）
Skubal(+12) >> Webb/Sánchez(+7) > Crochet/Skenes(+8) > Gilbert(+6) > Valdez(+5)

### RP 大幅貶值
- Cade Smith：8×8 VOR+9 → 7×7 VOR+6（IP -2 致命）
- 所有 RP 受 IP -2 結構性懲罰
- RP 從「前 4 輪必搶」降為「R5+ 隨緣」

### 打者陷阱/機會翻轉
- **翻身**：Cease(+1→+3)、Valdez(+3→+5)、Schwarber（punt 路線下可選）
- **崩盤**：Kwan(+4→+1)、Perdomo(+2→-1)、deGrom(+5→+2)
- **維持**：Soto(+10)、Judge(+9)、Vlad(+9)、Ramírez(+9)

### 守位稀缺性（維持）
2B（最稀缺）> SS > 1B ≈ 3B > C > OF

## 45 秒速查決策規則（7×7 版）

- **打者**：OPS > .850 → H+BB > 200 → K/AB < 20%，兩項通過就選
- **SP**：IP > 180 → ERA < 3.50，兩項通過就選
- **RP**：降低優先度，中後段隨緣
- **不確定時**：選高 PA 每日先發打者、高 IP 工作馬 SP
